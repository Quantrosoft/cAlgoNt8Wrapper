This software makes it possible to write one core trading algorithm 
and let it run on cTrader and NinjaTrader without any code changes in the core algorithm.

Installation:
To make this work there are 3 Visual Studio projects necessary in the following folders:
C:\Users\User\Documents\NinjaTrader 8\bin\Custom\Strategy\CoreAlgoName
C:\Users\User\Documents\cAlgo\Sources\Robots\CoreAlgoName
C:\Users\User\Documents\Source\cAlgoNt8Wrapper

The wrapper can be found here and must be cloned to C:\Users\User\Documents\Source\cAlgoNt8Wrapper
https://github.com/Quantrosoft/cAlgoNt8Wrapper

Samples for cTrader and NinjaTrader will follow soon.
