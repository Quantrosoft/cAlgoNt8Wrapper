This software makes it possible to write one core trading algorithm and let it run on cTrader and NinjaTrader without any code changes in the core algorithm.

# Installation
To make this work the following Visual Studio projects must be cloned to this folders:

File view:
| Folder Path                                                                                       | Description                                           |
|---------------------------------------------------------------------------------------------------|-------------------------------------------------------|
| `C:\Users\<User>\Documents\Source\cTraderNinjaTrader\cAlgoNt8Indicators`                          | Indicators used by cTrader and NinjaTrader sample project UltronNt |
| `C:\Users\<User>\Documents\Source\cTraderNinjaTrader\cAlgoNt8Wrapper`                             | The wrapper around the core algo                      |
| `C:\Users\<User>\Documents\Source\cTraderNinjaTrader\cTrader\UltronNt`                            | A simple trading bot compiled for cTrader             |
| `C:\Users\<User>\Documents\Source\cTraderNinjaTrader\NinjaTrader\UltronNt`                        | A simple trading bot compiled for NinjaTrader         |
