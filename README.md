This software makes it possible to write one core trading algorithm and let it run on cTrader and NinjaTrader without any code changes in the core algorithm.

# Installation
To make this work there are 3 Visual Studio projects necessary in the following folders:

| Repository | Clone to path |
|---|---|
| [cAlgoNt8Wrapper](https://github.com/Quantrosoft/cAlgoNt8Wrapper) | `C:\Users\<User>\Documents\Source\cAlgoNt8Wrapper` |
| [RoadToSuccess-cTrader](https://github.com/Quantrosoft/RoadToSuccess-cTrader) | `C:\Users\<User>\Documents\cAlgo\Sources\Robots\RoadToSuccess-cTrader` |
| [NinjaTrader.Custom](https://github.com/Quantrosoft/NinjaTrader.Custom) | `C:\Users\<User>\Documents\NinjaTrader 8\bin\Custom` |

Samples for cTrader and NinjaTrader will follow soon.
